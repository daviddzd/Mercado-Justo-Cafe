# Mercado-Justo-Cafe
Mercado Justo — Análisis y predicción de precios del café en el sureste mexicano (Chiapas, Oaxaca y Veracruz). Proyecto integrador de 5° semestre, Ciencias de Datos para Negocios, Universidad Nacional Rosario Castellanos. Incluye ETL con PySpark, pronóstico ARIMA, clasificación Random Forest, optimización lineal (PuLP), dashboard Plotly.
